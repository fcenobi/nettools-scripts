Simple tool for testing ping in bulk

Usage:	pingtest [filename] | [start ip address] [end ip address]
Example:	pingtest iplist.txt

Input file should include one ip address/hostname per line

Example output:

jason@host:~$ pingtest test
google.com UP
yahoo.com UP
github.com UP
8.8.8.8 UP
10.0.0.0 UP

jason@host:~$ pingtest 10.0.0.0 10.0.0.9
10.0.0.0 UP
10.0.0.1 UP
10.0.0.2 DOWN
10.0.0.3 DOWN
10.0.0.4 DOWN
10.0.0.5 DOWN
10.0.0.6 DOWN
10.0.0.7 DOWN
10.0.0.8 DOWN
